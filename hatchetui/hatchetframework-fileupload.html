图片上传

图片上传组件 - hatchetframework.fileupload

HTML

<!--js引入-->
<script src="hatchetframework/hatchetframework-fileupload/hatchetframework-fileupload.js"></script>

<!--图片上传按钮-->
<div class="hatchet-update">
    <a href="javascript:;">图片上传</a>
    <input type="file" id="up-img" />
</div>
<!--提示语-->
<span class="hatchet-update-tip">(只支持 .jpg .png)</span>
<!--上传图片容器-->
<div class="clear" id="img-show"></div>

JavaScript

    new Hatchet.fileupload(options);

参数

  • options 部分必填，部分可选。
参数 描述 类型 默认值
upImgBtn 上传图片ID string 必填
imgShow 上传图片存放路径 string 必填
ImgType 支持的上传图片格式 Array ["gif", "jpeg", "jpg", "bmp", "png"]
mode 上传图片模式 string 'change' 图片上传：'push'
string 上传图片样式 json {width:150,height:150}
callback 回调 Function null
