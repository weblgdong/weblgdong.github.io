JS

var Menu,MenuTab,map = new Map();
var options = {
    skin: 'blue-theme',
    dataUrl:'getMenuListByUserId.jhtml',
    tab: true
};

$(function () {
    Menu = new HatchetFramework.Menu("#navtree", options);
    MenuTab = new HatchetFramework.Tab();
});

基本参数

new HatchetFramework.Menu(id, options);

options - 配置参数

类型：json

属性 值
skin 'blue-theme'和'gray-theme'两种皮肤
dataUrl 数据链接
tab 是否显示顶部导航
